Stuffit Expander 4.0: The ultimate tool for decompressing files fron the 'net. It debinhexes, unstuffs, and more. With the 'Expander Enhancer' addon from Aladdin, it can deal with even more formats.

ShrinkWrap 1.4.2: If you've ever run into an Apple 'Disk Image' file, you'll love this program. It lets you mount disk image files on the desktop, just as if they were real floppies. It also creates, duplicates, and manipulates image files -- never touch DiskCopy again.

TechTool 1.0.8: TechTool is great for troubleshooting -- this one program completely rebuilds any Mac's Desktop file, zaps the entire PRAM, scans for damaged system files, and generates complete system profiles.

BBEdit Lite 3.5.1: If you read files from the Internet, create web pages, or just need a fast, powerful text editor, BBEdit is the program you're looking for. Flexible and blazingly fast, it's the little brother of Bare Bones Software's commercial program, BBEdit.
